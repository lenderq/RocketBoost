## Описание проекта

**Rocket Boost** — это аркадная физическая игра на Unity, где игрок управляет космическим кораблем, преодолевая препятствия с помощью тяги и вращения. Цель — добраться до финишной платформы, избегая столкновений с ландшафтом и движущимися объектами. Проект демонстрирует работу с физикой Rigidbody, новой системой ввода (Input System) и партиклами.

В репозитории доступен готовый билд игры в папке `FinalBuild`.

## Ключевые особенности

- **Физическое управление**: Движение основано на `Rigidbody.AddRelativeForce` для реалистичной инерции.
- **New Input System**: Использование `UnityEngine.InputSystem` для обработки ввода (тяга, вращение, дебаг).
- **Система коллизий**:
    - Обработка столкновений через теги (`Friendly`, `Finish`, `Untagged`).
    - Логика победы/поражения с задержкой (Invoke) и перезагрузкой сцен.
- **Визуальные и звуковые эффекты**:
    - Партиклы для двигателей (основной/боковые), взрывов и успеха.
    - Динамическое аудио: звук двигателя синхронизирован с нажатием клавиш.
- **Динамическое окружение**: Скрипт `Oscillator.cs` для создания подвижных препятствий через `Mathf.PingPong`.
- **Инструменты отладки**: Встроенные читы для тестирования (отключение коллизий, пропуск уровней).


## Управление

| Действие | Клавиша (Keyboard) | Описание |
| :-- | :-- | :-- |
| **Тяга (Thrust)** | `Space` / `W` | Активация главного двигателя (полет вверх) |
| **Вращение** | `A` / `D` или `←` / `→` | Поворот корабля влево/вправо |
| **Выход** | `Escape` | Закрыть приложение |
| **Debug: Next Level** | `L` | Мгновенный переход на следующий уровень |
| **Debug: No Clip** | `C` | Отключение столкновений |

## Техническая реализация

- **Movement.cs**: Основной контроллер. Использует `FixedUpdate` для физических расчетов и управляет стейтом аудио/партиклов в зависимости от `Input.IsPressed`.
- **CollisionHandler.cs**: Машина состояний для обработки конца игры. Блокирует управление при краше (`DisableMoving`) и управляет переходами между сценами.
- **Oscillator.cs**: Реализует логику подвижных платформ, интерполируя позицию между двумя точками (`Vector3.Lerp`).


## Установка и запуск

1. **Запуск билда**:
    - Перейдите в папку `FinalBuild/`.
    - Запустите исполняемый файл `.exe`.
2. **Открытие проекта в Unity**:
    - Клонируйте репозиторий: `git clone https://github.com/lenderq/RocketBoost.git`.
    - Откройте через Unity Hub.
    - Сцены находятся в папке `Assets/Scenes`.
